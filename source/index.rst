无网环境下的 Python 开发指南
========================================================

说起有关 Python 的指南，便一定得提到 K 神创建的『Python最佳实践指南』，英文名字是『The Hitchhiker’s Guide to Python!』。毫不夸张地说，K 神创建的指南算地上是一份完美的指南示例，优雅无比，涵盖了有关 Python 安装、配置和日常使用的全部内容，并带有强烈主观意见地推荐不同领域的 Python 框架，还提供非常多的学习资料，强烈推荐阅读。因为 K 神提供的指南实在过于优秀，我也便学着创建了这份指南，主要针对无网的工作环境，希望能够为无网环境下的 Python 开发人员提供一些参考，让 Python 能够进入更多的领域。
